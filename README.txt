
Mercy Clinics – OAC Calculator (PWA)
====================================

This Progressive Web App (PWA) calculates Oral Aperture Circumference (OAC) 
using IID (Inter-Incisor Distance) and ICD (Inter-Commissural Distance).

FORMULA:
OAC ≈ π × √((IID² + ICD²) / 2)

INSTALLATION:
1. Upload all files in this folder to a simple web server.
   Options:
     - GitHub Pages
     - Netlify
     - Any hospital intranet server
2. Once hosted, open the link in Chrome/Edge (mobile or desktop).
3. Use "Add to Home Screen" to install the app as a standalone PWA.
